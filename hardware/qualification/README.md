# Component Qualification Records

**Location for qualified component evidence.**

## Purpose

Per `docs/COMPONENT_QUALIFICATION_PROCESS.md` — no approved BOM entry without a record here (or explicit provisional authorization in work package).

## Record naming

```
[Manufacturer]_[MPN]_qualification.md
```

Example: `Infineon_BTS50085-1TMA_qualification.md` — **do not create until qualification task assigned**.
