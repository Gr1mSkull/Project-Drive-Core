# 007 — Component Selection

Компонентная база Gen1: automotive-grade где возможно, доступность на LCSC/Mouser/DigiKey, без экзотики.

Архитектура силовой части: **Smart High Side Switch** (Infineon PROFET+ / BTS).

## Logic Board

| Компонент | Выбор | Комментарий |
|-----------|--------|-------------|
| MCU | **STM32G474RET6** | CAN FD, FPU, ADC, таймеры |
| Конфиг | **FM24CL64B** (FRAM) | Практически бесконечный ресурс записи |
| Flash | **W25Q128JV** | Логи, резервные конфиги |
| Watchdog | **TPS3430** | Аппаратный, независимо от MCU |
| CAN ×2 | **TCAN1042HG** | CAN FD, ECU + Body |
| USB | USB-C + **TUSB320** | Настройка, отладка |
| Debug | SWD (Tag-Connect или разъём) | |

## Питание

| Компонент | Выбор |
|-----------|--------|
| TVS (load dump) | **SM8S36A** / SM8S33A |
| Reverse protection | **LM74700-Q1** |
| Buck 12→5 V | **TPS54560** / LM5164-Q1 |
| LDO 3.3 V | **TPS7A1633-Q1** |
| Power monitor | **TPS3702** |

## Power Board — Smart High Side

| Класс | Пример | Примечание |
|-------|--------|------------|
| 60 A | Infineon BTS7008 / BTS7200 class | EHPS, крупные потребители |
| 30 A | BTS500xx / BTS700xx | Вентиляторы, помпа |
| 15 A | PROFET+2 средний ток | Свет, ECU, средние нагрузки |
| 5 A | PROFET+2 малый ток | Салон, габариты |
| H-bridge ×2 | **BTN8982TA** | Стеклоподъёмники, stall detection |

Измерение тока: **INA228** (общий), встроенная диагностика PROFET (по каналам).

## Radio Board

Готовый модуль (не проектировать с нуля):

- **ESP32-S3-WROOM** или **ESP32-C6-WROOM**
- SPI к Logic Board
- Wi-Fi, BLE, Web UI, OTA

## Button Box

| Компонент | Выбор |
|-----------|--------|
| MCU | STM32G0 / G431 |
| CAN | TCAN1042 |
| RGB (опц.) | WS2812B / IS31 |

## Разъёмы

**Deutsch** — единый стандарт: DTM (сигналы), DT (средний ток), DTP (силовые).

## Корпус DCC

Алюминиевый (Hammond / CNC). Плата через термопрокладки — корпус = радиатор.

## Ориентир бюджета Gen1

| Уровень | Сумма |
|---------|-------|
| Минимальный | $180–250 |
| Реалистичный | $250–400 |
| С запасом | $400–700 |

Структура затрат: ~40–50% силовые ключи, ~20% питание, ~15% MCU, ~15% разъёмы/механика.
