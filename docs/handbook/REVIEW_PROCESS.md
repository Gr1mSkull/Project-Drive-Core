# Review Process

**Версия:** Handbook v1.0 · **Статус:** Proposed

## Rule

**Every change requires review.**

## Review checklist

| Area | Check |
|------|-------|
| Requirements | Scope matches task; no silent scope creep |
| Architecture | No unauthorized ADR-level changes |
| Tests | Relevant tests run or NOT VERIFIED stated |
| Docs | Affected docs and EDL updated |
| Risks | Open questions and assumptions reported |
