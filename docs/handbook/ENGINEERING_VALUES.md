# Engineering Values

**Версия:** Handbook v1.0 · **Статус:** Proposed

## Core values

1. **Truth over assumptions** — evidence before implementation.
2. **Evidence over confidence** — datasheet and measurement, not intuition.
3. **Architecture over implementation** — interfaces and decisions before code.
