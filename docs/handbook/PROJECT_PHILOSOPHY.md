# Project Philosophy

**Версия:** Handbook v1.0 · **Статус:** Proposed

DriveCore is an open modular automotive platform.

## Principles (summary)

- Hardware is fixed; behavior is configurable.
- Modular nodes: DCC, ECU, Button Box, expansion.
- Suitable for motorsport and custom vehicle applications.
- No vendor lock-in; open documentation and interfaces.
