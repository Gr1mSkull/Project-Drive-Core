# 000 — Project Vision

## Миссия

Создать открытую, модульную автомобильную платформу, а не просто PDM.

Архитектура должна быть масштабируемой, ремонтопригодной и конфигурируемой.

## Главный ориентир

> **Мы не делаем устройство. Мы создаём платформу.**

Каждое решение отвечает на вопрос: *«Поможет ли оно создать следующий модуль?»*

## Цель Gen1 (E30)

Максимально упростить проводку гоночного автомобиля, повысить надёжность и диагностируемость. Один центральный силовой хаб вместо реле, предохранителей и разрозненной логики.

## Принципы (никогда не нарушаются)

1. **Fail operational** — отказ второстепенных функций не останавливает автомобиль.
2. **Modular by design** — любой модуль можно заменить без изменения остальных.
3. **Configuration over code** — поведение определяется конфигурацией, а не перепрошивкой.
4. **Open documentation** — любое инженерное решение документируется.
5. **Test before track** — новая функция не попадает в автомобиль без проверки на стенде.

## Дополнительные принципы Gen1

- DCC управляет автомобилем, ECU — только двигателем.
- Использовать лучшие готовые технологии, не изобретая их заново.
- Один тип разъёма = один класс сигналов (Deutsch DTM/DT/DTP).
- Минимум 20% запаса каналов под расширение.
- Документация ведётся одновременно с разработкой, а не «после».

## Долгосрочная цель

Через несколько лет инженер должен иметь возможность сказать: *«Я хочу сделать модуль ABS для DriveCore»* — и подключиться без устных объяснений архитектуры.
